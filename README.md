# englishWeChatApp
关于英语学习的微信小程序前端代码

# 功能列表
- 单词查询（发音、例句等）
- 文章阅读
- 阅读过程中生词查询
- 题目联系（单选/判断）
- 错题联系
- 月度统计（阅读文章数、做题情况）
